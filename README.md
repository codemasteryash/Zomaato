# 📖 Overview

**Zomaato** recreates the modern food discovery experience inspired by Zomato.

This project focuses on building an attractive and responsive interface where users can explore restaurants, browse food collections, and enjoy a premium food-ordering style UI.

Built entirely using frontend technologies for practicing UI development and responsive design.
